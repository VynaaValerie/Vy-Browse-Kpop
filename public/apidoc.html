VyBrowse API

API berita K-pop real-time yang menggerakkan KWave Search Engine. Mengambil data dari Soompi, Koreaboo, dan Google News secara langsung. Respons JSON bersih — tanpa autentikasi. Siap digunakan untuk bot WhatsApp, bot Discord, website, dan berbagai integrasi lainnya.

URL DASAR https://your-domain.vercel.app

Real-time

Berita langsung dari 3+ sumber, di-scrape sesuai permintaan dengan cache 5 menit.

JSON Bersih

Skema konsisten, teks sudah di-decode, tanpa entitas HTML.

Tanpa Auth

Semua endpoint publik. Tidak perlu API key apapun.

Siap untuk Bot

Dioptimalkan untuk bot WhatsApp, Telegram, dan Discord.

Trending

12 kata kunci K-pop paling trending, diperbarui tiap 2 menit.

Cache Cerdas

Cache sisi server mencegah scraping berulang yang tidak perlu.

URL Dasar

Produksi & Lokal 
https://your-domain.vercel.app          # Setelah deploy ke Vercel
http://localhost:5000                      # Development lokal
Semua endpoint mendukung CORS dari origin mana saja. Tinggal tambahkan path endpoint ke URL dasar. Semua respons bertipe Content-Type: application/json.

Autentikasi

Tidak perlu autentikasi. Semua endpoint API sepenuhnya publik. Bisa langsung dipanggil dari kode frontend, bot, script, atau HTTP client apapun — tanpa header, token, atau API key.
Contoh — Tanpa Auth 
# Tidak perlu header Authorization
curl "https://your-domain.vercel.app/api/trending"

# CORS terbuka — bisa dipanggil langsung dari browser
fetch('/api/news').then(r => r.json()).then(console.log);

Rate Limit & Cache

Respons di-cache di sisi server. Permintaan identik dalam jendela cache akan langsung dikembalikan tanpa memicu scraping baru. Interval polling yang disarankan tercantum di bawah.

EndpointTTL CacheInterval Poll MinimumKeterangan
/api/search5 menitSesuai permintaan penggunaKey: query + kategori + halaman
/api/news3 menitSetiap 3 menitKey: kategori + halaman
/api/trending2 menitSetiap 2 menitDi-refresh otomatis oleh frontend
/api/categoriesStatisSekali sajaCache di sisi klien

Format Respons

Setiap respons adalah objek JSON. Field success selalu ada. Saat terjadi error, field error dan code disertakan.

Respons sukses

Skema Objek Artikel

Setiap item dalam array data pada endpoint pencarian dan berita adalah objek Artikel.

idstringID unik artikel (alfanumerik acak, 9 karakter)
titlestringJudul artikel, sudah di-decode HTML entity, teks biasa bersih
descriptionstringRingkasan atau kutipan singkat, maks ~300 karakter, teks biasa
thumbnailstringURL lengkap gambar cover (JPG/PNG). Bisa string kosong jika tidak tersedia
urlstringURL artikel asli di sumber (Soompi, Koreaboo, dll.)
sourcestringNama penerbit (contoh: "Soompi", "Koreaboo", "Deadline")
sourceIconstringSatu huruf kapital mewakili sumber (untuk fallback ikon)
categorystringSalah satu dari: Comeback MV Drama Idol Concert Scandal Fashion
dateISO stringWaktu publikasi format ISO 8601 (UTC). Gunakan new Date(article.date) untuk parse
timeAgostringWaktu relatif yang mudah dibaca: "2h ago", "3d ago"
Catatan: Konten artikel (judul, deskripsi) berasal dari sumber berbahasa Inggris (Soompi, Koreaboo, Google News). Teks ini tidak diterjemahkan karena merupakan konten berita asli dari penerbit.

Skema Objek Trending

rankintegerPosisi 1–12 dalam daftar trending
keywordstringKata kunci K-pop / nama artis yang sedang trending
trendstring"up" atau "stable" — arah tren
countintegerPerkiraan jumlah pencarian/mention (indikatif, bukan angka pasti)

Kode HTTP Status

200

OK

Permintaan berhasil. success: true, data dikembalikan.

400

Bad Request

Parameter wajib tidak diisi (misal: tidak ada q di /search).

404

Not Found

Path endpoint tidak ditemukan. Cek kembali URL kamu.

500

Server Error

Scraping gagal atau terjadi error internal. Coba lagi sebentar.

Integrasi JavaScript

Client VyBrowse Lengkap
// vybrowse-client.js — drop-in API client
const VyBrowse = {
  base: 'https://your-domain.vercel.app',

  async cari(query, category = '', page = 1) {
    const p = new URLSearchParams();
    if (query)    p.set('q', query);
    if (category) p.set('category', category);
    p.set('page', page);
    const res = await fetch(`${this.base}/api/search?${p}`);
    if (!res.ok) throw new Error(`HTTP ${res.status}`);
    return res.json();
  },

  async berita(category = '', page = 1, limit = 12) {
    const p = new URLSearchParams({ category, page, limit });
    const res = await fetch(`${this.base}/api/news?${p}`);
    return res.json();
  },

  async trending() {
    const res = await fetch(`${this.base}/api/trending`);
    return res.json();
  },

  async kategori() {
    const res = await fetch(`${this.base}/api/categories`);
    return res.json();
  }
};

// Cara pakai:
const hasil   = await VyBrowse.cari('aespa');
const comeback = await VyBrowse.berita('Comeback', 1, 5);
const trending = await VyBrowse.trending();

Integrasi Bot WhatsApp

Kompatibel dengan Baileys, whatsapp-web.js, atau library WhatsApp berbasis Node.js lainnya.

whatsapp-web.js + VyBrowse
const axios    = require('axios');
const BASE_URL = 'https://your-domain.vercel.app';

// Handler perintah — jalankan di dalam client.on('message')
async function handlePerintahKpop(msg) {
  const body = msg.body.trim();

  // !kpop aespa → cari berita
  if (body.startsWith('!kpop ')) {
    const query = body.slice(6).trim();
    const { data } = await axios.get(
      `${BASE_URL}/api/search?q=${encodeURIComponent(query)}`
    );
    if (!data.success || !data.data.length) {
      return msg.reply('Tidak ada hasil untuk: ' + query);
    }
    const top3 = data.data.slice(0, 3);
    const balasan = top3.map((a, i) =>
      `*${i+1}. ${a.title}*\n` +
      `${a.description}\n` +
      `[${a.source} · ${a.timeAgo}] ${a.category}\n` +
      `${a.url}`
    ).join('\n\n');
    return msg.reply(`*Berita KWave — ${query}*\n\n${balasan}`);
  }

  // !trending → kata kunci trending teratas
  if (body === '!trending') {
    const { data } = await axios.get(`${BASE_URL}/api/trending`);
    const list = data.data.slice(0, 10).map(t =>
      `${t.rank}. *${t.keyword}* ${t.trend === 'up' ? '↑' : '→'} ${t.count.toLocaleString('id-ID')}`
    ).join('\n');
    return msg.reply(`*K-pop Trending Sekarang*\n\n${list}`);
  }

  // !comeback → berita comeback terbaru
  if (body === '!comeback') {
    const { data } = await axios.get(
      `${BASE_URL}/api/news?category=Comeback&limit=3`
    );
    const list = data.data.map(a =>
      `• *${a.title}*\n  ${a.url}`
    ).join('\n\n');
    return msg.reply(`*Comeback Terbaru*\n\n${list}`);
  }
}

// Daftarkan perintah
client.on('message', async msg => {
  if (['!kpop', '!trending', '!comeback'].some(c => msg.body.startsWith(c))) {
    await handlePerintahKpop(msg);
  }
});

Integrasi Python

Python — requests
import requests

BASE = 'https://your-domain.vercel.app'

def cari_kpop(query, category='', page=1):
    params = {'q': query, 'category': category, 'page': page}
    res = requests.get(f'{BASE}/api/search', params=params, timeout=10)
    res.raise_for_status()
    return res.json()

def ambil_trending():
    res = requests.get(f'{BASE}/api/trending', timeout=10)
    return res.json()['data']

def ambil_berita(category='', limit=10):
    res = requests.get(f'{BASE}/api/news',
      params={'category': category, 'limit': limit}, timeout=10)
    return res.json()['data']

# Contoh penggunaan
hasil = cari_kpop('aespa', category='Comeback')
for artikel in hasil['data']:
    print(f"[{artikel['source']}] {artikel['title']}")
    print(f"  {artikel['url']}\n")

trending = ambil_trending()
for t in trending[:5]:
    print(f"#{t['rank']} {t['keyword']} ({t['count']:,} mention)")

# VyBrowse API — Referensi cURL

BASE="https://your-domain.vercel.app"

## 1. Cari berita berdasarkan kata kunci
curl -s "$BASE/api/search?q=aespa"

## 2. Cari dengan filter kategori
curl -s "$BASE/api/search?q=BTS&category=Concert"

## 3. Cari halaman 2
curl -s "$BASE/api/search?q=BLACKPINK&page=2"

## 4. Feed berita terbaru (semua kategori)
curl -s "$BASE/api/news"

## 5. Berita terbaru — Comeback saja
curl -s "$BASE/api/news?category=Comeback"

## 6. Berita MV — 5 per halaman, halaman 2
curl -s "$BASE/api/news?category=MV&page=2&limit=5"

## 7. Kata kunci trending
curl -s "$BASE/api/trending"

## 8. Daftar semua kategori
curl -s "$BASE/api/categories"

## 9. Pretty print dengan jq
curl -s "$BASE/api/search?q=aespa" | jq '.data[] | {title, source, category, timeAgo}'

## 10. Ambil judul saja
curl -s "$BASE/api/news" | jq -r '.data[].title'

## 11. Cek hasMore untuk paginasi
curl -s "$BASE/api/news?page=1" | jq '{total, hasMore, page}'
